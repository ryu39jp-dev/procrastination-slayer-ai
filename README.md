# procrastination-slayer-ai
# 論理的マウント・アナリスト (Logical Annoying Mentor)

「先延ばし癖」を根性論ではなく、「技術とロジック」で解決するための監視システムです。

## 概要
大学の課題や資格勉強の進捗を報告すると、AIが論理的にその妥当性を分析し、
「うざい口調」でマウントを取りながら行動を促します。

## 技術スタック (予定)
- **Infrastructure:** AWS (Lambda, DynamoDB, API Gateway, EventBridge)
- **AI:** Amazon Bedrock (Claude 3 / Llama 3)
- **Language:** Python
- **Notification:** Discord or LINE
